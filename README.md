### 3. **Course Management System** (`course_management/`)
An enterprise-grade course management application built with Java and Spring Boot.

**Tech Stack:**
- Backend: Java (Spring Boot)
- Build Tool: Maven
- Database: SQL-based
- Deployment: Docker
- Environment: Multi-configuration (dev, qa, prod)

**Features:**
- RESTful API endpoints
- Database migrations in `src/main/resources/db`
- Environment-specific configurations
- Dockerized deployment
- Compiled JAR application

**Getting Started:**
```bash
cd course_management
mvn clean install
mvn spring-boot:run
```

Or use the Maven wrapper:
```bash
./mvnw spring-boot:run
```

**Build Docker Image:**
```bash
docker build -t course-management:latest .
docker run -p 8080:8080 course-management:latest
```
